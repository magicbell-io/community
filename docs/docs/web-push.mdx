---
title: Implement Web Push Notifications
subtitle: Learn how to send web push notifications from your app
---

MagicBell's Inbox offers [web push notifications](https://www.magicbell.com/web-push) out of the box on Chrome, Firefox, and Safari.

<Note>
  After your users enable web push notifications, ensure that they also grant permissions
  in both the browser and operating system settings to allow push notifications.
</Note>
