## Prerequisites

You need to have:

- A MagicBell account. Sign up for free at [magicbell.com](http://magicbell.com)!
- Your project's API key and secret - these are available on the Settings page.

We encourage you to run examples in your terminal as you read this guide. You can install curl by running:

```shell hideHeader
brew install curl
```

You can explore all available API endpoints [here](/docs/rest-api/reference).
